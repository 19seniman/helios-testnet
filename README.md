## HELIOS TESTNET 

An automated bot for interacting transaction with  HELIOS TESTNET

##  Prerequisites 
~ Node.js (v18 or higher)
~ npm or yarn
~ Pharos Testnet wallet with private keys
~ (Optional) Proxy list in proxies.txt


## INSTALLATION

```
git clone
```
```
cd
```
```
npm install
```
```
nano pk.txt
```
~ Run The Script
```
node lim.js
```
